# AURON W — Watch Selling Website

A fully designed and responsive watch-selling ecommerce website built using **HTML, CSS, and JavaScript**.  
This project showcases a premium UI, smooth layout, elegant product display, and modern front-end structure.

---

## Features
- Clean and modern UI
- Responsive layout for desktops and mobiles
- Hero banner with premium watch preview
- Product collection section with multiple models
- Featured products with price and Add-to-Cart buttons
- Navigation bar with search functionality (UI only)
- Organized folder structure (HTML / CSS / JS / Images)

---

## Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript**
- FontAwesome (icons)
